# 图形推理复盘流程

基于刘义恒3+3体系，对图形推理错题进行结构化复盘，建立特征信号→条件反射的积累体系。

## Trigger Conditions

当用户提出以下内容时加载：
- "图形推理复盘" / "图推错题复盘"
- "刘义恒体系的复盘"
- "这道图推题怎么做"
- "帮我分析图推/图形推理"
- "特征信号速查表"

## 做题阶段（10秒破题法）

按6眼顺序逐级排除，每眼3秒内锁定或跳过：

```
第1眼：元素完全相同？→ 位置规律（平移/旋转翻转）
第2眼：图形很像、有重复线条？→ 样式规律（加减同异/黑白运算）
第3眼：有额外小标记（点/圈/三角）？→ 标记规律
第4眼：图形规则、对称、美观？→ 属性规律（对称性优先，再曲直/开闭）
第5眼：又乱又不像又不对称？→ 数量规律（强特征优先→面/线/点/素/笔画/角）
第6眼：一图多独立图形？→ 图形间位置（相离/包含/相交）
特殊分流：黑白格→黑白块专项 | 汉字→汉字专项 | 立体→空间重构
```

## 错题分析模板

| 字段 | 填什么 |
|------|--------|
| **来源** | 哪年哪套题第几题 |
| **第一眼特征** | 如实写下第一眼看到的 |
| **第一反应考点** | 第一反应判断的方向 |
| **实际考点** | 正确考点 |
| **10秒破题法定位** | 按6步走一遍，标出哪一步锁定的考点 |
| **错因勾选** | ①/②/③/④/⑤ |
| **卡在哪里** | 具体卡在哪个环节 |
| **秒杀规律** | 提炼成一句话 |
| **同类题联想** | 之前做过的同类题 |

### 错因说明

| 错因 | 说明 |
|------|------|
| ①特征信号没捕捉到 | 没看到关键特征（如五角星→笔画，日字→笔画） |
| ②考点方向定错 | 往A类想其实是B类 |
| ③细化不够 | 知道是数量规律但不知道数什么 |
| ④计算/验证失误 | 数错了格子/算错了步数/奇点没数对 |
| ⑤时间压力 | 其实会做但没时间细想 |

## 特征信号速查表

| 你看到 | 直接想 |
|--------|--------|
| 元素完全一样、只挪动/转向 | 位置规律 |
| 图形很像、重复线条 | 样式运算 |
| 五角星/日/田/圆相交 | 数量·交点笔画 |
| 直角三角形/十字架/矩形外框 | 数量·直角 |
| 图形对称、规则、美观 | 属性·对称性 |
| 只有2种元素 | 数量·换算（图1+图3=2×图2） |
| 图上有额外小标记 | 标记规律 |
| 一图多独立图形 | 图形间位置 |
| 黑白格 | 黑白块专项 |
| 汉字 | 汉字专项 |
| 生活实物图（伞/椅/字母） | 属性·生活化（曲直/开闭/部分数） |
| 每图都带一段弧 | 属性·弧度/角度走势 |
| 六图风格迥异、要分类 | 属性：对称→曲直→开闭 |
| 图形规则却有孤立线头 | 数量·端点数递增/减 |
| π / 缺口圆 / 辐射符 | 判开闭，且高度警惕判反 |
| 选项有个"外形超像"题干的 | 形状陷阱，回去验数量规律 |
| 大框套子图 | 数量·外框边数 vs 内部最大图形边数 |
| 满图乱线、有固定小图形(△○) | 标记规律·只看小图形的位置关系 |
| 看到△和○/两个小元素共存 | 判包含/相交/相离，而非"有没有" |
| 自己正盯着"整体轮廓"看 | 强制切换 → 去看局部关系 |

## 归档路径

```
考公笔记/行测/错题汇总/图形推理/
├── 位置规律/
├── 样式规律/
├── 标记规律/
├── 属性规律/
├── 数量规律/（面/线/点/笔画/角/元素/部分数/面积）
├── 图形间位置/
├── 黑白块/
├── 汉字/
├── 空间重构/
└── imgs/
    └── YYYY-MM-DD/
        ├── 1.png
        └── ...
```

### 文件名格式

```
考点-核心内容.md
```

例：`属性规律-对称轴等面数复合.md`、`数量规律-面-面数量恒为1.md`

### 图片归类规范

图形推理截图统一存放在 `imgs/YYYY-MM-DD/` 下，按答题日期分子目录，避免不同批次覆盖。

## 积累体系

每题提炼一条秒杀规律，格式：

> 看到 **X特征** → 直接 **Y做法**

存入相应归档文件的顶部。
