# 判断推理复盘流程

基于薛睿判断推理课程体系，对论证推理/形式逻辑/分析推理/类比推理错题进行结构化复盘。

## Trigger Conditions

当用户提出以下内容时加载：
- "判断推理复盘" / "做判断推理错题复盘"
- "薛睿体系的复盘"
- "论证推理错了一道题"
- "形式逻辑/真假话/假言命题复盘"
- "帮我分析这道判断推理题"

## 知识体系

### 四大模块

| 模块 | 包含内容 |
|------|---------|
| **论证推理** | 八大方法：二句式推理 / 由因推果 / 由果推因 / 秒杀思路 / 解释型 / 选非题型 / 引入型 / 结论推出 |
| **形式逻辑** | 逻辑符号+等价变换+真假话：假言命题 / 真假话问题 / 文氏图 / 结构类比 |
| **分析推理** | 排序题 / 匹配题 / 分组题 |
| **类比推理** | 因果关系等 |

## 做题顺序（薛睿框架）

```
提问句 → 结论引词 → 结论句 → 论据 → 脑中预设推理 → 对比选项
```

## 复盘三问（每道错题必问）

1. **考什么？** 定位到具体知识点（如：二句式推理-削弱、由果推因-归因削弱）
2. **题干逻辑链是什么？要什么？** 论据→结论，分清是削弱/加强/假设/结论推出
3. **有没有相似题？** 找同类题，归纳通用解法

## 四大错因体系

| 错因 | 说明 | 占比参考 |
|------|------|---------|
| ① 不理解题干推理 | 没有正确理解题干的逻辑结构（因果/对比/类比等） | ~40% |
| ② 不明白选项功能 | 看不懂选项在题干中的作用（加强/削弱/搭桥/拆桥等） | ~20% |
| ③ 忽略题干主题范围 | 选了范围不对的选项（太宽泛/太片面/主体不符） | ~20% |
| ④ 出题人挖坑 | 被偷换概念/混淆时态/绝对化表述等陷阱干扰 | ~20% |

### 每道错题文件结构

1. YAML头
2. 标题（`考点-核心内容`）
3. 导航链（`> [[做题总纲]] ← [[判断推理]]`）
4. `## 题目` — 完整题目原文 + ABCD选项
5. 我的答案 + 正确答案
6. **批注** — 用薛睿风格梳理逻辑链
7. **错因分析** — 勾选错因 + 卡点说明
8. **正确解法** — 分步骤
9. **干扰项拆解** — 逐项分析
10. **秒杀规律** — 一句话提炼
11. **同类题联想** — [[wikilink]]

### 核心规律提炼

- 削弱归因 → 找"另有他因"力度 > "否定因果"
- 选非题 → 先判断哪些能削/能加强，排除后剩下的就是答案
