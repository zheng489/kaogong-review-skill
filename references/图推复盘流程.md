# 图推复盘流程

> [[做题总纲]] ← [[图形推理]]
>
> 基于刘义恒3+3体系
> 每练完一组题（建议10-20题）复盘一次

---

## 一、做题阶段（10秒破题法）

按6眼顺序逐级排除，每眼3秒内锁定或跳过：

```
第1眼：元素完全相同？→ 位置规律（平移/旋转翻转）
第2眼：图形很像、有重复线条？→ 样式规律（加减同异/黑白运算）
第3眼：有额外小标记（点/圈/三角）？→ 标记规律
第4眼：图形规则、对称、美观？→ 属性规律（对称性优先，再曲直/开闭）
第5眼：又乱又不像又不对称？→ 数量规律（强特征优先→面/线/点/素/笔画/角）
第6眼：一图多独立图形？→ 图形间位置（相离/包含/相交）
特殊分流：黑白格→黑白块专项 | 汉字→汉字专项 | 立体→空间重构
```

---

## 二、复盘阶段（错题分析模板）

每道错题/卡题记录以下内容：

| 字段 | 填什么 |
|------|--------|
| **来源** | 哪年哪套题第几题 |
| **第一眼特征** | 如实写下第一眼看到的（如"有黑白格"、"图形都对称"） |
| **第一反应考点** | 第一反应判断的方向（即使错了也要写） |
| **实际考点** | 正确考点 |
| **10秒破题法定位** | 按6步走一遍，标出哪一步锁定的考点 |
| **错因勾选** | ①特征信号没捕捉到 / ②考点方向定错 / ③细化不够 / ④计算验证失误 / ⑤时间压力 |
| **卡在哪里** | 具体卡在哪个环节 |
| **秒杀规律** | 提炼成一句话，如"看到日/田/五角星→数奇点判笔画" |
| **同类题联想** | 之前做过的同类题，建立题型档案 |

### 错因详细说明

| 错因 | 说明 |
|------|------|
| ①特征信号没捕捉到 | 没看到关键特征（如五角星→笔画，日字→笔画） |
| ②考点方向定错 | 往A类想其实是B类（如以为是对称性其实是数量） |
| ③细化不够 | 知道是数量规律但不知道数什么（没细分到面形状/三角形个数） |
| ④计算/验证失误 | 数错了格子/算错了步数/奇点没数对 |
| ⑤时间压力 | 其实会做但没时间细想 |

## 三、归档体系（Obsidian文件结构）

```
考公笔记/行测/错题汇总/图形推理/
├── 位置规律/（平移/旋转翻转/）
├── 样式规律/
├── 标记规律/
├── 属性规律/（对称/曲直/开闭...）
├── 数量规律/（面/线/点/笔画/角/元素/部分数/面积）
├── 图形间位置/
├── 黑白块/
├── 汉字/
├── 空间重构/
└── imgs/
    ├── YYYY-MM-DD/   ← 按答题日期分目录
    └── ...
```

### 文件名格式

```
考点-核心内容.md
```

### 图片归类规范

图形推理的题目截图统一存放在 `imgs/` 目录下，**按日期分子目录存放**，避免多批次题目使用相同编号导致覆盖。

```
图形推理/
└── imgs/
    ├── YYYY-MM-DD/     ← 按答题日期分目录
    │   ├── 1.png
    │   ├── 2.png
    │   └── ...
    └── ...
```

图片引用方式（推荐）：

```markdown
![[imgs/YYYY-MM-DD/编号.png]]
```

或备选：

```markdown
![](考公笔记/行测/错题汇总/图形推理/imgs/YYYY-MM-DD/编号.png)
```

新增题目时的操作步骤：
1. 把截图放入 `imgs/` 下对应的日期目录（没有则新建）
2. 在错题文件中引用 `![[imgs/YYYY-MM-DD/编号.png]]`
3. 不要使用纯数字平铺（如 `imgs/4.png`），不同批次会覆盖

## 四、积累体系（特征信号→条件反射）

每题提炼**一条秒杀规律**存入归档文件，格式：

> 看到 **X特征** → 直接 **Y做法**

### 特征信号速查表

| 你看到 | 直接想 |
|------|--------|
| 元素完全一样、只挪动/转向 | 位置规律 |
| 图形很像、重复线条 | 样式运算 |
| 五角星/日/田/圆相交 | 数量·交点笔画 |
| 直角三角形/十字架/矩形外框 | 数量·直角 |
| 图形对称、规则、美观 | 属性·对称性 |
| 只有2种元素 | 数量·换算（图1+图3=2×图2） |
| 图上有额外小标记 | 标记规律 |
| 一图多独立图形 | 图形间位置 |
| 黑白格 | 黑白块专项 |
| 汉字 | 汉字专项 |
| 生活实物图（伞/椅/字母） | 属性·生活化（曲直/开闭/部分数） |
| 每图都带一段弧 | 属性·弧度/角度走势 |
| 六图风格迥异、要分类 | 属性：对称→曲直→开闭 |
| 图形规则却有孤立线头 | 数量·端点数递增/减 |
| π / 缺口圆 / 辐射符 | 判开闭，且高度警惕判反 |
| 选项有个"外形超像"题干的 | 形状陷阱，回去验数量规律 |
| 大框套子图 | 数量·外框边数 vs 内部最大图形边数 |
| 满图乱线、有固定小图形(△○) | 标记规律·只看小图形的位置关系 |
| 看到△和○/两个小元素共存 | 判包含/相交/相离，而非"有没有" |
| 自己正盯着"整体轮廓"看 | 强制切换 → 去看局部关系 |

## 五、复盘频率建议

| 阶段 | 频率 | 重点 |
|------|------|------|
| 初期 | 每道错题必复盘 | 建立特征信号库的条件反射 |
| 中期 | 只复盘卡了60s+的题 | 精细化区分易混淆考点 |
| 后期 | 每周集中复盘一次 | 归类整理自己的易错题型档案 |
