# 判断推理复盘流程

> [[做题总纲]] ← [[判断推理]]
>
> 基于薛睿判断推理课程体系
> 每练完一组题（建议 20-40 题）复盘一次

---

## 一、做题顺序（薛睿框架）

```
提问句 → 结论引词 → 结论句 → 论据 → 脑中预设推理 → 对比选项
```

---

## 二、知识体系

### 四大模块

| 模块 | 包含内容 | 子题型 |
|------|---------|--------|
| **论证推理** | 八大方法 | 二句式推理 / 由因推果 / 由果推因 / 秒杀思路 / 解释型 / 选非题型 / 引入型 / 结论推出 |
| **形式逻辑** | 逻辑符号+等价变换+真假话 | 假言命题 / 真假话问题 / 文氏图 / 结构类比 |
| **分析推理** | 排序/匹配/分组/综合 | 排序题 / 匹配题 / 分组题 |
| **类比推理** | 因果关系等 | — |

---

## 三、复盘三问（每道错题必问）

| 问题 | 回答 |
|------|------|
| **考什么？** | 定位到具体知识点（如：二句式推理-削弱、由果推因-归因削弱） |
| **题干逻辑链是什么？要什么？** | 论据→结论，分清是削弱/加强/假设/结论推出 |
| **有没有相似题？** | 找同类题，归纳通用解法 |

---

## 四、四大错因体系

| 错因 | 说明 | 占比参考 |
|------|------|---------|
| **① 不理解题干推理** | 没有正确理解题干的逻辑结构（因果/对比/类比等） | ~40% |
| **② 不明白选项功能** | 看不懂选项在题干中的作用（加强/削弱/搭桥/拆桥等） | ~20% |
| **③ 忽略题干主题范围** | 选了范围不对的选项（太宽泛/太片面/主体不符） | ~20% |
| **④ 出题人挖坑** | 被偷换概念/混淆时态/绝对化表述等陷阱干扰 | ~20% |

## 五、错题记录格式

### 归档文件名格式

```
题型-核心内容.md
```

例：`二句式推理-削弱HR辅助作用.md`、`由果推因-气候变冷冲突质疑.md`、`假言命题-话剧角色分配.md`

### YAML头格式

```yaml
---
日期: YYYY-MM-DD
科目: 行测
模块: 判断推理
题型: 具体题型
错因: ①/②/③/④
---
```

### 每道错题文件结构

1. YAML头（日期/科目/模块/题型/错因）
2. 标题（`考点-核心内容`）
3. 导航链（`> [[做题总纲]] ← [[判断推理]]`）
4. **## 题目** — 完整题目原文 + ABCD选项
5. 我的答案 + 正确答案
6. **批注** — 用薛睿风格梳理逻辑链
7. **错因分析** — 勾选错因 + 卡点说明
8. **正确解法** — 分步骤
9. **干扰项拆解** — 逐项分析
10. **秒杀规律** — 一句话提炼
11. **同类题联想** — [[wikilink]]

- 削弱归因 → 找"另有他因"力度 > "否定因果"

## 七、归档路径

```
考公笔记/行测/错题汇总/判断推理/
├── 论证推理/
│   ├── 01_二句式推理/       ← 通用削弱/加强/反驳
│   ├── 02_由因推果/         ← 原因→结果推理
│   ├── 03_由果推因/         ← 归因类(最大模块)
│   ├── 04_秒杀思路/         ← 因果倒置等
│   ├── 05_解释型/           ← 待补充
│   ├── 06_选非题型/         ← 不能削/不能支持
│   ├── 07_引入型/           ← 前提假设
│   └── 08_结论推出/         ← 从题干推出结论
├── 形式逻辑/
│   ├── 01_假言命题/
│   ├── 02_真假话问题/
│   ├── 03_文氏图/
│   ├── 04_结构类比/
│   └── 05_逻辑符号与等价变换/
├── 分析推理/
└── 类比推理/
    └── 因果关系/
```

每道题文件命名示例：
```
论证推理/01_二句式推理/二句式推理-削弱HR辅助作用.md
论证推理/03_由果推因/由果推因-气候变冷冲突质疑.md
形式逻辑/02_真假话问题/真假话问题-四句话一真盗窃案.md
```

汇总索引页放 `考公笔记/行测/复盘/YYYYMMDD_判断推理错题复盘.md`
