# 言语理解复盘流程

> [[做题总纲]] ← [[言语理解]]
> 
> 基于郭熙言语课程体系
> 每练完一组题（建议15-25题）复盘一次

---

## 一、做题流程

```
读问题 → 读文段 → 预设答案 → 对比选项
```

### 文段阅读要点

1. **勾关键标志**：转折（但/然而/事实上）、因果（因此/所以）、对策（必须/应该/需要）、并列（同时/此外/另一方面）
2. **结构分层**：总分、分总、转折、并列、递进
3. **主体判定**：文段一直围绕谁在说？选项主语对不对？

---

## 二、知识体系

### 第一章 片段阅读

| 题型 | 细分 | 核心方法 |
|------|------|---------|
| **中心理解题** | 转折关系 | 转折后是重点，但注意多个转折的层级 |
| | 并列关系 | 并列两方面都要概括，共同变量法 |
| | 行文脉络·尾句观点 | `对此/这些……共同……/可以说`等总结词 |
| | 行文脉络·对策 | `必须/应该/因此/如何……是关键`，注意对策双向覆盖 |
| | 行文脉络·总分首句 | 首句即观点，后文展开 |
| **细节判断题** | 选是题 | 警惕绝对化词（决定/必须/所有） |
| | 选非题 | 偷换概念/混淆时态/过度推断 |
| **词句理解题** | 代词指代 | 就近原则，理清替代者与被替代者关系 |
| | 成语指代 | 理清谁替代谁，比喻的本体是什么 |
| **意图判断题** | 问题→对策 | 尾句对策>问题描述，具体对策>原理 |
| | 观点提炼 | 转折后是观点，注意选项程度不能超越原文 |
| | 反面论证 | `如果缺乏→就要有` |
| | 实验结论类 | 实验对比得出的结论才是核心 |
| **标题选择题** | 事件类 | 首句导语概括 |
| | 观点类 | 尾句对策做标题，选"怎么做"不选"别变成什么" |

### 第二章 语句表达

| 题型 | 核心方法 |
| --------- | -------------------------------- |
| **接语选择题** | 尾句引出新话题→下文讲什么；尾句已完结→前文总结 |
| **语句填空题** | 首句横线→总起后文；句中横线→承上启下；尾句横线→前文逻辑推结论 |
| **语句排序题** | 首句定位（总结句/指代词不能做首句）→话题捆绑→顺序验证 |

（暂无错题积累）

## 三、逐题复盘模板

每道错题记录以下内容：

| 字段 | 填什么 |
|------|--------|
| **来源** | 哪年哪套题第几题 |
| **题型** | 如：中心理解·后对策 / 细节判断·选是题 |
| **文段结构** | 转折/并列/总分/对策等 |
| **我的答案** |  |
| **正确答案** |  |
| **错因勾选** | ①不理解题干推理 / ②不明白选项功能 / ③忽略主题范围 |
| **批注** | 用花生/郭熙风格梳理文段逻辑链 |
| **干扰项拆解** | 逐项分析为什么错 |
| **同类题规律** | 提炼通用解法 |

### 错因详细说明

| 错因 | 说明 | 典型表现 |
|------|------|---------|
| ①不理解题干推理 | 对文段结构/逻辑判断错 | 代词指代搞反、并列当转折、因果关系搞混 |
| ②不明白选项功能 | 看懂了文段但选不对选项 | 选项程度超越原文（"别夸大"→"否定"）、混淆概念区分vs定义、绝对化词 |
| ③忽略主题范围 | 选了范围不对的选项 | 只覆盖对策一半、主体不符、扩大/缩小范围 |

### 归档文件名格式

```
题型-核心内容.md
```

例：`中心理解-文化强国中外并蓄.md`、`细节判断-社交媒体偏见因素.md`、`语句排序-古典园林以静为韵律.md`

### YAML头格式


---

## 六、复盘频率建议

| 阶段 | 频率 | 重点 |
|------|------|------|
| 初期 | 每道错题必复盘 | 建立文段结构识别能力 |
| 中期 | 只复盘做错的题 | 重点攻选项辨析（程度/主体/范围） |
| 后期 | 每周集中复盘一次 | 归类整理易错题型档案 |
